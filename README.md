Pour utiliser les scripts, il faut d'abord ce rendre dans le dossier scripts:

`cd scripts`

Ensuite, il faut donner les droits d'exécution aux scripts:

`chmod +x *.sh`

Enfin, il suffit d'exécuter le script setup.sh:

`./setup.sh`
